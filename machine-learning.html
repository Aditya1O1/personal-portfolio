New York Taxi Fare Prediction

Built a machine learning regression model to predict taxi fare amounts in New York City using ride-related attributes such as coordinates, time, and passenger count. The project was based on the Kaggle competition dataset and aimed at improving fare estimation accuracy for ride-hailing platforms.

  • Preprocessed over 1 million rows to clean and engineer time & location features.
  • Extracted pickup hour, day, month from datetime and filtered NYC-bound coordinates.
  • Trained XGBoost Regressor to learn complex fare patterns.
  • Achieved a Root Mean Squared Error (RMSE) of 4.30 on the test set.
  • Deployed prediction model on cleaned test data and exported submission file for evaluation.
